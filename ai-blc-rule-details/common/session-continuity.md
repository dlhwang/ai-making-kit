# Session Continuity

When resuming an existing AI-BLC workflow, read `blc-docs/blc-state.md` first.

## Resume Summary

Present a short summary:

- Work item:
- Current phase:
- Current stage:
- Last completed stage:
- Next recommended step:
- Open questions:

## Context Loading

Load relevant prior artifacts before continuing:

- Context artifacts for planning stages
- Requirements before execution stages
- Execution plan before deliverable creation
- Quality criteria before review
- Review summary before operations

## Resume Options

If the next step is unclear, create `blc-docs/resume-questions.md` using the question format guide.

## Audit

Log resume activity in `blc-docs/audit.md`, including the detected state and selected next step.
