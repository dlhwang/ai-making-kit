# Quality Criteria

Purpose: define how the work will be judged before review.

## Criteria

Use criteria relevant to the work:

- Accuracy
- Completeness
- Source traceability
- Confidentiality
- Compliance
- Readability
- Decision usefulness
- Executive readiness
- Operational usability
- Timeliness

## Output

Create `blc-docs/execution/quality-criteria.md`.

Use these criteria in Review and Validation.
