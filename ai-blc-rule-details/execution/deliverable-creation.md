# Deliverable Creation

Purpose: create the approved business deliverables.

## Rules

- Follow the approved execution plan.
- Respect scope in and scope out.
- Make assumptions visible.
- Identify source, evidence, or uncertainty for important claims.
- Do not invent facts.
- Preserve open questions.
- Use the requested format and tone.

## Output

Create deliverables in `blc-docs/execution/deliverables/` unless the user asks for another location.

Also create or update `blc-docs/execution/deliverables/deliverable-index.md` with:

- Deliverable name
- Purpose
- Audience
- Status
- Review needs
