# 공학적 메이킹 품질 게이트

## 목적

품질 게이트는 다음 단계로 넘어가기 전에 반드시 통과해야 하는 기준입니다.

## Gate 0: 문제 정의

통과 조건:

- 만들 물건의 목적이 명확하다.
- 사용자가 누구인지 정해졌다.
- 성공 상태가 정의되었다.
- 주요 제약이 적혔다.

실패 예:

- “예쁘게 만들기”
- “대충 맞게 만들기”
- “나중에 치수 맞추기”

## Gate 1: 재료/공법 확정

통과 조건:

- 재료가 정해졌다.
- 실제 두께를 측정했다.
- 절단 방법이 정해졌다.
- 접착/체결 방법이 정해졌다.
- 허용 오차를 가정했다.

실패 예:

- 재료가 바뀌었는데 CAD가 그대로다.
- 두께를 카탈로그 값으로만 쓴다.

## Gate 2: 실측 완료

통과 조건:

- 기준 물체의 핵심 치수를 측정했다.
- 케이블/부속품 치수를 측정했다.
- 움직이는 부분의 여유를 확인했다.
- 가정값과 실측값이 분리되어 있다.

실패 예:

- 실제 부품을 끼워보지 않고 전체 제작한다.

## Gate 3: 제작 구조 검토

통과 조건:

- 부품을 실제로 조립할 수 있다.
- 접착면/체결면이 있다.
- 조립 중 위치를 잡는 기준이 있다.
- 좌우/앞뒤가 구분된다.
- 초보자가 실수할 지점이 줄어들었다.

실패 예:

- 슬롯은 있는데 탭이 없다.
- 접착해야 하는데 접착 위치가 없다.

## Gate 4: CAD 출력 검증

통과 조건:

- 3D preview가 열린다.
- 2D cut layout이 export된다.
- 테스트 쿠폰이 export된다.
- 개별 부품 모드가 export된다.
- 출력 파일을 여는 도구에서 정상 표시된다.

실패 예:

- 3D 미리보기만 있고 제작 도면이 없다.
- DXF가 3D extrusion 상태다.

## Gate 5: 테스트 쿠폰 통과

통과 조건:

- 탭/슬롯 fit이 맞다.
- 핵심 구멍이 맞다.
- 접착 후 변형이 허용 범위다.
- 실제 부품이 들어간다.
- 조정할 파라미터를 기록했다.

실패 예:

- 전체 부품을 먼저 자른다.
- fit 실패 원인을 기록하지 않는다.

## Gate 6: 1차 프로토타입 통과

통과 조건:

- 핵심 기능이 동작한다.
- 실제 사용 동작이 가능하다.
- 안정성이 허용된다.
- 케이블/움직임 간섭이 없다.
- 접착/체결이 버틴다.

실패 예:

- 모양은 맞지만 사용할 때 넘어짐.
- 부품은 들어가지만 탈착이 불편함.

## Gate 7: 초보자 제작성 통과

통과 조건:

- 부품 번호가 있다.
- 수량과 적층 수가 명확하다.
- 조립 순서가 있다.
- 접착 위치가 설명되어 있다.
- 실패 시 조정표가 있다.
- 초보자가 테스트 쿠폰부터 시작하게 되어 있다.

실패 예:

- CAD만 있고 설명서가 없다.

## Gate 8: 최종 패키지 통과

통과 조건:

- CAD 원본
- 제작용 CAD
- cut layout
- novice layout
- test coupon
- assembly guide
- measured specs
- validation report
- known risks

이 모두가 있다.

## 게이트 운영 원칙

각 게이트에서 실패하면 다음 단계로 넘어가지 않는다.

특히 다음 세 가지는 건너뛰면 안 된다.

- 재료 두께 실측
- 테스트 쿠폰
- 초보자 조립 가이드
