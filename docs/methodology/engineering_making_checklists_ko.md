# 공학적 메이킹 체크리스트

## 1. 시작 체크리스트

- [ ] 만들 물건의 목적이 한 문장으로 정의되었다.
- [ ] 실제 사용자가 누구인지 정했다.
- [ ] 제작자의 숙련도를 정했다.
- [ ] 사용할 재료를 정했다.
- [ ] 사용할 공구/가공법을 정했다.
- [ ] 실패해도 되는 테스트 범위를 정했다.

## 2. 실측 체크리스트

- [ ] 기준 물체의 전체 치수를 쟀다.
- [ ] 돌출부 치수를 쟀다.
- [ ] 케이블/커넥터 치수를 쟀다.
- [ ] 움직이는 부품의 여유 공간을 쟀다.
- [ ] 재료 두께를 여러 지점에서 쟀다.
- [ ] 절단 kerf를 측정했다.
- [ ] fit 여유를 기록했다.

## 3. 재료 체크리스트

- [ ] 재료명
- [ ] 실제 두께
- [ ] 두께 편차
- [ ] 결/방향성
- [ ] 접착 가능성
- [ ] 휘어짐 가능성
- [ ] 최소 절단 반경
- [ ] 반복 사용 내구성

## 4. 기능 체크리스트

- [ ] 대상 물체를 잡거나 지지한다.
- [ ] 사용자가 쉽게 넣고 뺄 수 있다.
- [ ] 케이블이나 부속품이 눌리지 않는다.
- [ ] 사용 중 흔들리거나 넘어지지 않는다.
- [ ] 반복 사용 시 헐거워지지 않는다.
- [ ] 청소/수리/교체가 가능하다.

## 5. 제작 구조 체크리스트

- [ ] 부품 수가 과도하지 않다.
- [ ] 같은 부품의 적층 수가 명확하다.
- [ ] 좌우 부품이 헷갈리지 않는다.
- [ ] 접착면이 충분하다.
- [ ] 조립 중 각도를 잡을 방법이 있다.
- [ ] 탭/슬롯이 실제 재료 두께에 맞는다.
- [ ] 접착제가 구멍을 막지 않는다.
- [ ] 공구가 들어갈 공간이 있다.

## 6. CAD 체크리스트

- [ ] 파라미터 이름이 명확하다.
- [ ] 실측값과 가정값이 분리되어 있다.
- [ ] 3D assembly와 2D cut layout이 분리되어 있다.
- [ ] fit_check 모드가 있다.
- [ ] test_coupon 모드가 있다.
- [ ] novice layout이 필요하면 따로 있다.
- [ ] 시각용 dummy가 실제 제작 부품과 구분된다.
- [ ] CLI export가 검증되었다.

## 7. 테스트 쿠폰 체크리스트

- [ ] 대표 탭
- [ ] 대표 슬롯
- [ ] 핵심 구멍
- [ ] 곡선 절단 샘플
- [ ] 케이블/부품 통과 샘플
- [ ] 접착 테스트 조각
- [ ] 판정 기준

## 8. 프로토타입 체크리스트

- [ ] 핵심 fit만 먼저 확인했다.
- [ ] 전체 접착 전에 임시 조립했다.
- [ ] 실제 사용 물체를 대보았다.
- [ ] 케이블/움직임을 확인했다.
- [ ] 안정성을 확인했다.
- [ ] 실패 지점을 기록했다.
- [ ] 수정 파라미터를 정했다.

## 9. 초보자 문서 체크리스트

- [ ] 부품 번호표
- [ ] 부품 수량표
- [ ] 적층 수량표
- [ ] 필요한 도구
- [ ] 조립 순서
- [ ] 접착 위치
- [ ] 건조/압착 방법
- [ ] fit 확인 시점
- [ ] 실패 시 조정표

## 10. 최종 출고 체크리스트

- [ ] 원본 CAD 보존
- [ ] 제작용 CAD 정리
- [ ] DXF/SVG/PDF 출력 파일 생성
- [ ] 테스트 쿠폰 파일 생성
- [ ] 조립 가이드 작성
- [ ] 검증 결과 기록
- [ ] 변경 이력 기록
- [ ] 미해결 위험 기록
