# Xiaomi Watch S4 하드보드지 도킹스테이션 조립 가이드

## 대상

이 문서는 하드보드지를 잘라 Xiaomi Watch S4 충전 도킹스테이션을 만드는 절차입니다.

초보자는 전체 도면을 바로 자르지 말고 반드시 테스트 쿠폰부터 자르십시오.

## 사용할 SCAD 파일

```text
xiaomi_watch_s4_cardboard_dock_hardboard.scad
```

## 권장 출력 순서

### 1. 테스트 쿠폰

```powershell
& 'C:\Program Files\OpenSCAD\openscad.com' -D 'view_mode=8' -o test_coupon.dxf xiaomi_watch_s4_cardboard_dock_hardboard.scad
```

먼저 이것만 잘라서 다음을 확인합니다.

- 하드보드지 탭이 슬롯에 들어가는가
- 너무 빡빡하거나 헐겁지 않은가
- 충전기 둥근 모서리가 맞는가
- 스트랩 슬롯 크기가 괜찮은가
- 케이블 채널 폭이 충분한가

### 2. 초보자용 전체 레이아웃

```powershell
& 'C:\Program Files\OpenSCAD\openscad.com' -D 'view_mode=9' -o novice_cut_layout.dxf xiaomi_watch_s4_cardboard_dock_hardboard.scad
```

초보자는 `view_mode=9`를 권장합니다. 사이드 패널 내부 구멍을 제거해 손절단과 접착이 더 쉽습니다.

### 3. 일반 전체 레이아웃

```powershell
& 'C:\Program Files\OpenSCAD\openscad.com' -D 'view_mode=1' -o cut_layout.dxf xiaomi_watch_s4_cardboard_dock_hardboard.scad
```

일반 레이아웃은 사이드 패널 내부 구멍이 있어 더 가볍지만 제작 난이도가 조금 높습니다.

## 적층 접착 원칙

하드보드지는 얇은 판을 겹쳐 두께를 만듭니다.

기본값:

- 전면/후면/사이드 패널: 2장 적층
- 베이스: 3장 적층

접착 방법:

1. 같은 부품끼리 모서리를 정확히 맞춥니다.
2. 접착제를 아주 얇게 바릅니다.
3. 슬롯과 구멍 안쪽으로 접착제가 흘러들지 않게 합니다.
4. 클립으로 잡거나 무거운 책을 올립니다.
5. 충분히 마를 때까지 기다립니다.

주의:

- 접착제를 많이 바르면 하드보드지가 휘어집니다.
- 구멍이 어긋나면 충전기와 탭이 맞지 않습니다.
- 베이스 슬롯은 막히면 안 됩니다.

## 테스트 쿠폰 판정

### 탭/슬롯

좋은 상태:

- 손으로 밀면 들어간다.
- 들어간 뒤 흔들림이 크지 않다.
- 하드보드지가 찢어지지 않는다.

너무 빡빡하면:

- `slot_fit`을 키웁니다.
- 예: `0.45`에서 `0.65`.

너무 헐거우면:

- `slot_fit`을 줄입니다.
- 예: `0.45`에서 `0.25`.

### 충전기 구멍

좋은 상태:

- 충전기가 무리 없이 들어간다.
- 너무 덜컥거리지 않는다.
- 앞면이 약간 앞으로 나올 수 있다.

너무 빡빡하면:

- `charger_clearance`를 키웁니다.
- 모서리는 사포로 조금씩 다듬습니다.

너무 헐거우면:

- 충전기 주변에 얇은 종이나 테이프를 보강하거나 `charger_clearance`를 줄입니다.

## 조립 순서

### 1. 적층 부품 만들기

먼저 다음 부품을 각각 적층합니다.

- A1 + A2 = 전면 패널
- B1 + B2 = 후면 패널
- C1 + C2 = 좌측 사이드 패널
- D1 + D2 = 우측 사이드 패널
- E1 + E2 + E3 = 베이스

완전히 마르기 전에는 다음 단계로 가지 마십시오.

### 2. 베이스와 사이드 패널 결합

1. 좌측 사이드 패널의 탭을 베이스 슬롯에 끼웁니다.
2. 우측 사이드 패널도 같은 방식으로 끼웁니다.
3. 기울어지지 않았는지 확인합니다.
4. 탭 주변에 소량의 접착제를 바릅니다.
5. 각도 게이지 J로 양쪽 패널이 같은 각도인지 확인합니다.

### 3. 전면 패널 붙이기

1. 전면 패널을 사이드 패널 앞쪽 기준면에 맞춥니다.
2. 충전기 구멍이 위쪽, 스트랩 슬롯이 아래쪽입니다.
3. 각도 게이지 J로 기울기를 맞춥니다.
4. 사이드 패널과 만나는 선에 얇게 접착합니다.
5. 마를 때까지 움직이지 않게 잡습니다.

### 4. 후면 패널 붙이기

1. 후면 패널을 전면 패널 뒤쪽에 배치합니다.
2. 케이블 구멍이 충전기 높이와 맞아야 합니다.
3. 케이블 채널은 아래쪽으로 내려갑니다.
4. 사이드 패널에 얇게 접착합니다.

### 5. 충전기 spacer ring 붙이기

1. G 부품을 전면 패널의 충전기 구멍 뒤쪽에 맞춥니다.
2. 충전기를 임시로 끼워 위치를 확인합니다.
3. 충전기가 비뚤어지지 않으면 G를 접착합니다.

### 6. 워치 saddle 붙이기

1. H 부품은 전면 패널 앞쪽에 붙입니다.
2. 워치 하단을 받치는 말굽형 부품입니다.
3. 워치를 올려보고 중심이 맞는지 확인한 뒤 접착합니다.

### 7. 전면 낮은 턱 붙이기

1. F 부품을 베이스 앞쪽에 붙입니다.
2. 워치가 앞으로 미끄러지는 것을 막는 보조 턱입니다.
3. 너무 높거나 방해되면 붙이기 전에 위치를 조금 조정합니다.

### 8. 충전기와 케이블 정리

1. 충전기를 전면 구멍에 넣습니다.
2. 케이블을 후면 구멍과 채널로 빼냅니다.
3. 케이블이 꺾이지 않는지 확인합니다.
4. 후면 retainer I를 붙여 충전기가 뒤로 빠지지 않게 합니다.

주의:

- 충전기를 완전히 고정하기 전에 반드시 실제 충전이 되는지 확인하십시오.
- 충전이 안 되면 retainer를 붙이지 말고 충전기 앞뒤 위치를 조정하십시오.

## 최종 확인

완성 후 다음을 확인합니다.

- 워치를 한 손으로 올려놓을 수 있는가
- 충전 표시가 뜨는가
- 스트랩이 걸리지 않는가
- 케이블이 심하게 꺾이지 않는가
- 책상 위에서 흔들리지 않는가
- 워치 화면을 터치해도 도크가 넘어지지 않는가

## 실패했을 때 조정할 값

| 문제 | 조정값 |
| --- | --- |
| 탭이 안 들어감 | `slot_fit` 증가 |
| 탭이 너무 헐거움 | `slot_fit` 감소 |
| 충전기가 안 들어감 | `charger_clearance` 증가 |
| 충전기가 흔들림 | `charger_clearance` 감소 또는 종이 shim 추가 |
| 스트랩이 걸림 | `strap_slot_w`, `strap_slot_h` 증가 |
| 케이블이 눌림 | `cable_channel_w`, `cable_bend_clearance` 증가 |
| 도크가 가벼움 | `base_layers` 증가 또는 바닥 보강 |

## 초보자 권장 방식

처음 만드는 경우:

1. `test_coupon.dxf`만 먼저 자릅니다.
2. 치수가 맞으면 `novice_cut_layout.dxf`를 자릅니다.
3. 모든 부품에 연필로 번호를 씁니다.
4. 적층 부품만 먼저 붙이고 하루 말립니다.
5. 다음 날 본 조립을 합니다.
