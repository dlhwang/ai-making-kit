# Xiaomi Watch S4 하드보드지 도킹스테이션 설계 보완 검토

## 결론

현재 `xiaomi_watch_s4_cardboard_dock.scad`는 좋은 개념 설계이지만, 아직 하드보드지로 바로 제작 가능한 최종 설계도는 아닙니다.

사용자가 원하는 것은 샤오미 Watch S4를 매일 쉽게 올려놓고 충전할 수 있는 도킹스테이션입니다. 따라서 다음 버전은 단순한 미리보기 모델이 아니라, 실제 제작과 사용을 견디는 구조로 바뀌어야 합니다.

핵심 방향:

- 충전기가 항상 같은 위치에 고정되어야 합니다.
- 워치가 자연스럽게 충전 위치로 안내되어야 합니다.
- 스트랩이 걸리거나 밀어내지 않아야 합니다.
- 케이블이 꺾이지 않고 빠져야 합니다.
- 하드보드지 부품은 조립 기준면이 명확해야 합니다.
- 절단 레이아웃은 실제 부품 제작용 2D 도면이어야 합니다.

## 현재 설계에서 확인된 값

SCAD 기준 확인값:

| 항목 | 값 |
| --- | ---: |
| 하드보드지 두께 가정 | `3.0 mm` |
| 도크 폭 | `70 mm` |
| 도크 깊이 | `86 mm` |
| 도크 높이 | `105 mm` |
| 패널 기울기 파라미터 | `62 deg` |
| 워치 지름 가정 | `47.3 mm` |
| 충전기 가정 크기 | `39.5 x 39.5 x 9.8 mm` |
| 충전기 포켓 개구부 | `40.4 x 40.4 mm` |
| 스트랩 슬롯 | `28 x 9 mm` |
| 케이블 구멍 | `12 mm` |
| 케이블 채널 | `7 mm` |

사용자 확정 조건:

- 재료는 하드보드지입니다.
- 목표 제품은 Xiaomi Watch S4 충전 도킹스테이션입니다.

## 제작 가능성 평가

현재 설계는 주요 패널 형상은 있습니다. 하지만 조립 구조가 닫혀 있지 않습니다.

부족한 점:

- 슬롯은 있으나 대응하는 돌출 탭이 없습니다.
- 충전기를 정확히 앞뒤로 잡아주는 stop 기준이 없습니다.
- 워치 본체를 받쳐주는 실제 크래들 형상이 없습니다.
- `cut_layout()`은 얇은 3D 미리보기이고 순수 2D 절단 도면이 아닙니다.
- 파일 주석/인코딩이 불안정해서 원본을 바로 대규모 수정하기 어렵습니다.

하드보드지는 작은 마찰 체결보다 넓은 접착면과 큰 기준 탭이 유리합니다. 그러므로 첫 실사용 버전은 순수 끼워맞춤보다 `접착 보조 탭 구조`가 맞습니다.

## 사용자 편의성 평가

### 워치 올려놓기

현재 구조는 충전기 위치를 보여주지만, 워치가 자동으로 중심에 잡히는 물리적 안내면이 없습니다.

보완:

- 충전기 주변에 얕은 말굽형 또는 하단 반원형 가이드를 추가합니다.
- 워치 하단을 받치는 낮은 saddle/ledge를 둡니다.
- 위쪽은 열어 두어 한 손으로 쉽게 들어올릴 수 있게 합니다.

### 충전 접촉

충전기 앞면이 패널과 정확히 어떤 관계인지 정해져 있지 않습니다.

보완:

- `charger_face_offset` 파라미터를 추가합니다.
- 하드보드지에서는 충전기 앞면이 패널보다 약간 앞으로 나오게 하는 편이 좋습니다.
- 시작값은 `0.3~0.8 mm` 정도의 조정 가능한 돌출량이 적절합니다.

### 스트랩 처리

현재 스트랩 슬롯은 `28 x 9 mm`입니다.

보완:

- 실제 스트랩 폭과 두께를 측정해야 합니다.
- 슬롯 폭은 `스트랩 폭 + 4 mm` 정도로 시작합니다.
- 슬롯 높이는 `스트랩 두께 + 4 mm`로 잡되, 손절단이면 최소 `10~12 mm`를 권장합니다.
- 슬롯 양끝은 둥글게 처리해 찢어짐을 줄입니다.

### 케이블 처리

현재 케이블 채널은 `7 mm`입니다.

보완:

- 케이블 직경 + 최소 `3 mm` 여유를 둡니다.
- 충전기 뒤쪽에 케이블이 꺾이지 않을 bend relief 공간을 둡니다.
- 커넥터를 통과시켜야 한다면 현재 채널은 부족할 가능성이 큽니다.

## 구조 보강 권고

### 베이스

현재 바닥은 `70 x 86 x 3 mm` 단일 판입니다.

하드보드지로는 가벼워서 워치를 터치할 때 밀리거나 넘어질 수 있습니다.

보완:

- 바닥을 2겹으로 설계합니다.
- 뒤쪽 깊이를 늘릴 수 있게 파라미터화합니다.
- 바닥 아래 미끄럼 방지 패드나 종이 마찰 패드를 붙일 공간을 둡니다.

### 사이드 프레임

현재 사이드 패널은 삼각 지지대 역할을 합니다.

보완:

- 사이드 패널을 조립 기준 프레임으로 격상합니다.
- 전면/후면 패널이 사이드 프레임에 기대거나 끼워지는 기준면을 둡니다.
- 하단은 베이스에 큰 탭으로 꽂고 접착합니다.

### 전면 패널

현재 전면 패널은 충전기 구멍과 스트랩 슬롯이 있습니다.

보완:

- 충전기 구멍을 둥근 사각형으로 변경합니다.
- 구멍 주변 최소 잔여 폭을 확인합니다.
- 워치 하단 받침을 별도 부품 또는 전면 패널 부착 부품으로 둡니다.

### 후면 패널

현재 후면 패널은 케이블 구멍과 채널이 있습니다.

보완:

- 케이블이 나가는 방향을 명확히 합니다.
- 충전기를 뒤에서 누르는 retainer strip을 추가합니다.
- 충전기 탈착형이면 후면 retainer를 분리 가능한 구조로 둡니다.

## 다음 SCAD 설계 방향

새 파일로 시작하는 것을 권장합니다.

```text
xiaomi_watch_s4_cardboard_dock_hardboard.scad
```

원본은 보존합니다.

새 설계의 모드:

```text
assembly
fit_check
cut_layout_2d
cut_layout_preview
front_panel
back_panel
side_left
side_right
base
```

추가할 파라미터:

```scad
material_t = 3.0;
kerf = 0.2;
slot_fit = 0.4;
glue_gap = 0.2;
charger_corner_r = 4;
charger_face_offset = 0.5;
cable_d = 4.0;
cable_bend_clearance = 10;
```

이 값들은 확정값이 아니라 하드보드지 1차 시제품용 시작값입니다. 실제 측정 후 바꿔야 합니다.

## 우선순위

1. 원본을 보존하고 UTF-8 안전한 새 SCAD 파일을 만듭니다.
2. 하드보드지 기준 파라미터를 정리합니다.
3. 충전기 구멍을 둥근 사각형으로 바꿉니다.
4. 충전기 앞면 돌출량과 후면 retainer를 추가합니다.
5. 워치 하단 saddle/guide를 추가합니다.
6. 사이드 프레임과 베이스의 실제 탭/접착 구조를 추가합니다.
7. `cut_layout_2d()`를 만들어 실제 절단 도면으로 쓸 수 있게 합니다.
8. OpenSCAD CLI로 각 모드를 검증합니다.
9. 작은 slot/tab 테스트 쿠폰을 먼저 절단합니다.
10. 전면 패널만 먼저 잘라 충전기와 워치 fit을 확인합니다.

## 현재 단계의 설계 판단

최종 설계는 `끼워 맞추는 장난감 같은 구조`보다 `접착면이 넓고 조립 기준이 명확한 구조`가 맞습니다.

하드보드지에서 가장 중요한 것은 정밀한 작은 탭이 아니라:

- 부품이 쉽게 정렬되는가
- 접착할 면이 충분한가
- 충전기가 움직이지 않는가
- 워치를 한 손으로 올리고 뺄 수 있는가
- 케이블이 자연스럽게 빠지는가

이 기준으로 보면, 현재 설계의 다음 단계는 치수 몇 개를 고치는 것이 아니라 제작 구조를 재설계하는 것입니다.
