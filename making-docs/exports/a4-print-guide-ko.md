# A4 출력 후 손재단 가이드

대상 파일: `xiaomi_watch_s4_cardboard_dock_hardboard_v2.scad`

이 가이드는 A4 용지에 1:1로 출력한 종이를 하드보드지 위에 올려 밑그림을 그리고 칼로 자르는 방식 기준입니다.

## 제일 중요한 출력 설정

프린터 설정에서 반드시 아래처럼 고르세요.

- 용지: A4
- 배율: 100%
- 실제 크기
- 페이지 맞춤, 용지에 맞게 축소, 자동 맞춤: 끄기

출력 후 각 템플릿 위쪽의 `50mm scale check` 선을 자로 재세요. 이 선이 50mm가 아니면 그 출력물은 쓰면 안 됩니다.

## 먼저 출력할 파일

폴더:

`making-docs/exports/a4-templates/`

출력 파일:

| 순서 | 파일 | 용도 |
| --- | --- | --- |
| 1 | `a4_00_cut_list.svg` | 전체 수량표 |
| 2 | `a4_01_test_coupon.svg` | 먼저 잘라보는 테스트 조각 |
| 3 | `a4_02_front_rear.svg` | 전면판/후면판 |
| 4 | `a4_03_base.svg` | 바닥판 3장 |
| 5 | `a4_04_side_panels.svg` | 왼쪽/오른쪽 측판 |
| 6 | `a4_05_small_parts.svg` | 작은 부품 전체 |

## 실제로 잘라야 하는 수량

1.5mm 하드보드지 기준입니다.

| 부품 | 잘라야 하는 수량 | 어디에 쓰는가 |
| --- | ---: | --- |
| 전면판 | 2장 | 충전기와 시계가 보이는 앞판 |
| 후면판 | 2장 | 케이블이 지나가는 뒷판 |
| 왼쪽 측판 | 2장 | 왼쪽 삼각 지지대 |
| 오른쪽 측판 | 2장 | 오른쪽 삼각 지지대 |
| 바닥판 | 3장 | 아래 받침 |
| 앞 턱 | 2장 | 시계/도크가 앞으로 미끄러지지 않게 잡는 낮은 턱 |
| 충전기 링 | 4장 | 충전기 주변을 두껍게 잡아주는 사각 링 |
| 시계 받침 | 2장 | 시계 아래쪽을 받치는 둥근 받침 |
| 후면 고정판 | 2장 | 뒤쪽에서 충전기/케이블을 잡는 작은 판 |
| 하부 보강대 | 2장 | 뒤쪽 아래를 보강하고 케이블 길을 잡는 판 |
| 각도 게이지 | 선택 1장 | 조립할 때 앞판 각도를 맞추는 임시 도구 |
| 풀 펴개 | 선택 1장 | 풀을 얇게 펴는 임시 도구 |

## 어떤 페이지를 몇 번 따라 그리나

| 템플릿 | 작업 |
| --- | --- |
| `a4_01_test_coupon.svg` | 1번만 자릅니다. 본 제작 전에 꼭 먼저 해보세요. |
| `a4_02_front_rear.svg` | 전면판 그림을 2번, 후면판 그림을 2번 따라 그립니다. |
| `a4_03_base.svg` | 바닥판 3개가 이미 들어 있습니다. 전부 1개씩 자릅니다. |
| `a4_04_side_panels.svg` | 페이지 전체를 2번 따라 그립니다. 왼쪽 2장, 오른쪽 2장이 됩니다. |
| `a4_05_small_parts.svg` | 그려진 작은 부품을 전부 자릅니다. |

## 추천 작업 순서

1. `a4_01_test_coupon.svg`를 먼저 출력해서 자릅니다.
2. 충전기, 케이블, 탭 구멍이 너무 빡빡한지 확인합니다.
3. 괜찮으면 나머지 A4 템플릿을 출력합니다.
4. 출력물을 하드보드지에 테이프로 고정합니다.
5. 모든 바깥선과 안쪽 구멍을 연필로 옮깁니다.
6. 안쪽 구멍부터 자르고, 마지막에 바깥선을 자릅니다.
7. 같은 부품끼리 먼저 겹쳐서 맞는지 확인합니다.
8. 바닥판 3장, 전면판 2장, 후면판 2장처럼 같은 부품끼리 풀로 붙입니다.
9. 충전기와 케이블을 끼워보고 잘 맞을 때만 최종 조립합니다.

## 조립 전 체크

- 충전기 링 4장을 겹쳤을 때 충전기가 억지로 들어가면 안 됩니다.
- 바닥판 3장의 측판 끼우는 홈 위치가 서로 어긋나면 안 됩니다.
- 측판은 왼쪽 2장, 오른쪽 2장이 필요합니다. 같은 방향만 4장 만들면 조립이 안 됩니다.
- 풀칠 전에 반드시 충전기와 케이블을 실제로 넣어보세요.
