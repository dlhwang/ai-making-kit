# Prototype Plan

## Build Sequence

1. Cut test coupon.
2. Tune SCAD parameters from coupon results.
3. Cut one front panel layer, one rear panel layer, one side panel, and one base layer as a partial prototype if material is limited.
4. Dry-fit charger and cable before gluing.
5. Cut full layout only after the interface fit passes.

## Do Not Glue Before

- Charger opening has been tested.
- Cable path has been tested.
- Side panels sit flat in base slots.
- The watch reaches the charger face without strap interference.

## Data to Record

| Data | Use |
| --- | --- |
| Charger insertion force | Tune charger fit. |
| Cable exit bend | Tune rear relief/channel. |
| Side-panel wobble | Tune tab slots and glue order. |
| Watch seating position | Tune saddle and strap slot. |

## Expected Revisions

- Clearance tuning after first coupon.
- Cable channel enlargement if cable is thick or stiff.
- Dock width/depth adjustment if the watch/strap makes the dock unstable.
