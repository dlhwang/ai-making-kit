# Quality Criteria

## Accuracy

- Source-derived values must match the SCAD or prior analysis documents.
- Unknown physical values must be marked as unknown.

## Completeness

The review must cover:

- Material behavior.
- Structural stability.
- Charger retention.
- Watch retention.
- Cable routing.
- Strap routing.
- Assembly method.
- Cutting/export method.

## Source Traceability

Claims should identify whether they come from:

- The SCAD source.
- Prior generated analysis docs.
- User-provided constraints.
- Engineering inference.

## Usability

Recommendations must favor:

- Easy insertion/removal of the watch.
- Stable charging contact.
- Cable that exits cleanly.
- Assembly that a normal user can perform without precision machinery, unless a cutter is later specified.

## Operational Usability

The final plan should be directly usable for the next SCAD revision.
