# Xiaomi Watch S4 하드보드지 도킹스테이션 - 신입생 제작 난이도 평가

## 평가 질문

갓 공학도로 입학한 20살 신입생에게 다음 두 가지만 주어진다면 완성할 수 있는가?

- `xiaomi_watch_s4_cardboard_dock_hardboard.scad`
- 하드보드지

## 결론

현재 상태 그대로라면 `완성 가능성은 있지만 난이도는 높음`입니다.

평가 등급:

```text
CAD/설계 구조: B
하드보드지 제작 적합성: B-
초보자 완성 가능성: C
초보자 설명 친화성: D+
실사용 도킹 가능성: B- before fit test
```

즉, 설계 방향은 맞지만 아직 “초보자가 설계도만 보고 완성할 수 있는 키트”는 아닙니다. 조립 설명서, 부품 라벨, 적층 순서, 접착 기준면, 테스트 쿠폰이 추가되어야 합니다.

## 현재 설계가 좋아진 점

### 1. 하드보드지 적층 개념이 들어갔다

현재 새 SCAD는 다음 구조를 사용합니다.

```scad
sheet_t = 1.5;
wall_layers = 2;
base_layers = 3;
wall_t = sheet_t * wall_layers;
base_t = sheet_t * base_layers;
```

이 방향은 좋습니다. 얇은 하드보드지를 여러 장 겹쳐 벽체와 베이스 강성을 만드는 방식은 초보자도 이해하기 쉽고, 재료 오차에도 비교적 강합니다.

### 2. 2D 절단 레이아웃이 생겼다

`view_mode = 1`에서 DXF 출력 가능한 2D 레이아웃이 나옵니다. 이것은 이전 설계보다 큰 진전입니다.

### 3. 충전기/워치 사용성이 반영됐다

추가된 요소:

- 둥근 충전기 개구부
- 충전기 spacer ring
- 후면 retainer
- 워치 하단 saddle
- 넓어진 스트랩 슬롯
- 케이블 bend relief

이 요소들은 실제 도킹스테이션으로 쓰기 위한 방향과 맞습니다.

## 초보자가 막힐 지점

### 1. 부품 이름이 절단 도면에 표시되지 않는다

현재 `cut_layout_2d()`는 형상을 나열하지만, 각 부품이 무엇인지 도면 안에 표시하지 않습니다.

초보자는 다음을 구분하기 어렵습니다.

- 전면 패널 1층/2층
- 후면 패널 1층/2층
- 좌측 사이드/우측 사이드
- 베이스 1층/2층/3층
- spacer ring
- saddle
- rear retainer
- front lip

평가:

```text
초보자 실패 가능성: 높음
```

개선:

- 절단 도면 외부에 `PART A`, `PART B` 라벨용 작은 기준표를 만들 것.
- SCAD 내부 텍스트는 cutter 호환성이 떨어질 수 있으므로, 별도 PDF/Markdown 조립 안내서로 보완하는 편이 안전합니다.

### 2. 적층 순서가 명확하지 않다

설계는 벽체 2겹, 베이스 3겹을 전제로 합니다. 하지만 초보자는 같은 모양을 왜 여러 장 잘라야 하는지, 어느 면끼리 붙여야 하는지 모를 수 있습니다.

특히:

- 베이스 슬롯이 있는 층을 전부 같은 방향으로 붙여야 하는지
- 슬롯이 막히지 않게 접착제를 어떻게 발라야 하는지
- 전면 패널 두 장의 구멍을 정확히 맞추는 방법
- spacer ring과 saddle을 어느 면에 붙이는지

이 설명이 없으면 정렬 오차가 커집니다.

평가:

```text
초보자 실패 가능성: 높음
```

개선:

- `assembly_guide.md`가 필요합니다.
- 적층 부품은 `클립으로 고정 -> 모서리 맞춤 -> 얇게 접착 -> 무게 올려 건조` 순서가 명시되어야 합니다.

### 3. 접착 기준면이 도면상으로는 부족하다

설계 철학은 glue-assisted입니다. 하지만 실제 설계도만 보면 어디에 풀을 발라야 하는지 명확하지 않습니다.

문제:

- side panel과 base의 탭은 있지만, 전면/후면 패널이 side frame에 어떻게 접착되는지 초보자에게 직관적이지 않습니다.
- front lip 위치는 3D 모델에서는 보이지만 2D 절단물만 보고는 위치 기준을 알기 어렵습니다.
- rear retainer가 후면 어느 위치에 붙는지 설명이 필요합니다.

평가:

```text
초보자 실패 가능성: 중간-높음
```

개선:

- 조립 지그 역할을 하는 간단한 `angle gauge` 또는 `side alignment template`을 추가하는 것이 좋습니다.
- 베이스 위에 전면/후면 패널 위치 기준선을 표시할 수 있는 별도 template이 필요합니다.

### 4. 하드보드지 손절단 난이도가 높을 수 있다

둥근 사각형, charger ring, saddle은 기능적으로 좋지만 손칼로 자르면 초보자에게 난이도가 있습니다.

어려운 형상:

- 충전기 둥근 사각 구멍
- spacer ring
- horseshoe saddle
- side panel 내부 삼각 cutout
- base의 좁은 슬롯

평가:

```text
레이저/DXF 절단: 중간 난이도
손칼 절단: 높음
```

개선:

- 손절단 버전에서는 내부 lightening cutout을 제거하는 것이 좋습니다.
- 충전기 구멍은 처음부터 완성 치수로 자르기보다 작게 자르고 사포/줄로 맞추는 방식이 안전합니다.
- base slot은 초보자에게 어려우므로 slot 폭을 더 넓히고 접착 의존도를 높이는 편이 낫습니다.

### 5. 충전기 실제 치수 검증 전에는 fit 실패 가능성이 있다

현재 값은 합리적인 시작값이지만 실제 충전기 모서리 반경, 케이블 위치, 스트랩 치수가 확정되지 않았습니다.

초보자에게 바로 전체 절단을 시키면 다음 문제가 날 수 있습니다.

- 충전기가 너무 빡빡함
- 충전기가 너무 헐거움
- 케이블이 후면에서 눌림
- 워치가 충전기 중심에 안 맞음
- 스트랩이 슬롯에서 걸림

평가:

```text
실사용 fit 실패 가능성: 중간
```

개선:

- 반드시 `test_coupon`을 먼저 만들어야 합니다.
- 본체 전체를 자르기 전에 충전기 구멍, 스트랩 슬롯, 탭/슬롯만 시험해야 합니다.

## 현재 난이도 평가

### 도구별 난이도

| 제작 방식 | 난이도 | 초보자 완성 가능성 |
| --- | --- | --- |
| 레이저 커터 + DXF | 중간 | 가능 |
| 커팅 플로터 | 중간-높음 | 가능하지만 세팅 필요 |
| 프린트 템플릿 + 손칼 | 높음 | 실패 가능성 큼 |
| SCAD만 보고 수작업 치수 재기 | 매우 높음 | 비권장 |

### 초보자 기준 작업 난이도

| 작업 | 난이도 | 이유 |
| --- | --- | --- |
| DXF 생성 | 중간 | 명령어를 알아야 함. |
| 부품 구분 | 높음 | 도면 라벨 없음. |
| 적층 접착 | 중간-높음 | 정렬과 건조 압착 필요. |
| 탭/슬롯 조립 | 중간 | 치수 오차에 민감. |
| 충전기 fit 조정 | 높음 | 실제 부품을 보며 수정해야 함. |
| 최종 조립 | 중간-높음 | 각도와 위치 기준이 부족함. |

## 20살 신입생에게 줄 수 있는가?

조건부로 가능합니다.

가능한 조건:

- DXF를 대신 뽑아주거나 생성 명령을 알려준다.
- 절단은 레이저 커터나 정밀 커터를 쓴다.
- 부품별 라벨이 있는 조립 안내서를 제공한다.
- test coupon을 먼저 자르게 한다.
- 접착제, 집게, 자, 칼, 사포, 무거운 책 같은 보조 도구를 같이 준다.

불가능하거나 위험한 조건:

- SCAD 파일과 하드보드지만 준다.
- 부품 라벨이 없다.
- 조립 순서가 없다.
- 손칼로 모든 곡선과 슬롯을 처음부터 정확히 자르라고 한다.
- 실제 충전기/워치 치수를 확인하지 않는다.

## 최종 난이도 판정

현재 그대로:

```text
신입생 단독 완성 난이도: 7.5 / 10
성공 확률: 45~60%
```

레이저 커팅 DXF와 조립 가이드가 제공될 경우:

```text
신입생 단독 완성 난이도: 4.5 / 10
성공 확률: 75~85%
```

테스트 쿠폰과 라벨, 접착 순서, 사진/그림 안내까지 제공될 경우:

```text
신입생 단독 완성 난이도: 3 / 10
성공 확률: 85~90%
```

2. 부품 이름표와 적층 수량표
3. test coupon 전용 2D 모드

6. 접착 순서와 건조/압착 안내
7. 충전기 fit 조정 절차

## 설계 수정 권고

### 1. 초보자 모드를 추가한다

SCAD에 다음 모드를 추가하는 것이 좋습니다.

```text
8 test_coupon_2d
9 novice_cut_layout_2d
```

`novice_cut_layout_2d`에서는 내부 lightening cutout을 제거하고, 어려운 곡선 부품 수를 줄입니다.

### 2. 부품 라벨용 별도 문서를 만든다

DXF에 직접 글자를 넣는 것보다 Markdown/PDF로 부품 번호표를 제공하는 편이 안정적입니다.

예:

```text
A1/A2: Front panel layers
B1/B2: Back panel layers
C1/C2: Left side layers
D1/D2: Right side layers
E1/E2/E3: Base layers
F: Front lip
G: Charger spacer ring
H: Watch saddle
I: Rear retainer
```

### 3. 조립 지그를 추가한다

초보자는 각도 맞추기가 어렵습니다. 하드보드지 조각으로 만드는 간단한 angle gauge가 있으면 성공률이 크게 올라갑니다.

### 4. 테스트 쿠폰을 필수화한다

전체 절단 전에 다음만 자르는 테스트 모드가 필요합니다.

- base slot
- side tab
- charger rounded corner sample
- strap slot sample
- cable channel sample

## 공학적 결론

이 설계는 공학적으로 방향이 좋고, 하드보드지 적층 구조도 타당합니다. 하지만 현재 산출물은 “제작 가능한 CAD”이지 “초보자가 바로 완성 가능한 키트”는 아닙니다.

신입생에게 주려면 설계 난이도를 낮추는 것이 아니라 `실수할 수 있는 지점을 제거하는 정보 설계`가 필요합니다.

다음 보완의 핵심은 CAD 형상보다:

- 부품 식별,
- 적층 순서,
- 접착 위치,
- 조립 기준,
- 테스트 쿠폰,
- 초보자용 단순 절단 레이아웃

입니다.
