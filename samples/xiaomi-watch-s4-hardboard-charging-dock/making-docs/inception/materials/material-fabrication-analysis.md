# Material and Fabrication Analysis

## Material Assumptions

| Parameter | Current value | Status |
| --- | --- | --- |
| Sheet material | Hardboard/cardboard | Assumed |
| Sheet thickness | 1.5 mm | Assumed from existing hardboard design |
| Wall stack | 2 layers | Design assumption |
| Base stack | 3 layers | Design assumption |

## Fabrication Method

Primary target: hand cutting or laser cutting from sheet stock.

## Tolerances

| Tolerance | Current value | Validation |
| --- | --- | --- |
| Kerf | 0.20 mm | Must be measured with coupon. |
| Tab slot fit | 0.45 mm | Must be measured with coupon. |
| Charger fit clearance | 0.70 mm | Must be measured with real charger. |
| Glue gap | 0.25 mm | Assembly assumption. |

## Joining Method

Use PVA glue, wood glue, or contact adhesive over broad faces. Tabs are treated as registration aids, not primary structural fasteners.

## Design Implications

- Large glue faces are preferred over small interlocking features.
- Rounded slots reduce tearing and overcut sensitivity.
- Stacked layers should be dry-fit before permanent glue.
- Side-panel lightening cutouts are optional; beginner layout keeps them solid.

## Unknowns Requiring Test Coupon

- Actual sheet thickness.
- Actual kerf.
- Whether the charger opening needs more or less clearance.
- Whether the cable channel is wide enough for the real cable.
