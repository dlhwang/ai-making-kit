# Work Units Generation

Execute for multi-part, multi-owner, or multi-prototype making work.

## Break Down By

- Measurement tasks
- CAD tasks
- Fabrication tasks
- Test coupon tasks
- Prototype build tasks
- Documentation tasks
- Review tasks
- Release package tasks

## Output

Create `making-docs/inception/plans/work-units.md`.

For each unit include:

- Owner/role
- Input
- Output
- Dependency
- Review gate
- Done criteria
