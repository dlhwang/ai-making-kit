# Measurement Plan

Purpose: define what must be measured before final fabrication.

## Analyze

- Critical external dimensions
- Internal fit dimensions
- Cable/connector dimensions
- Moving part clearances
- Material thickness
- Kerf
- Bend radius
- Weight or center-of-mass needs
- Required ergonomic measurements

## Output

Create `making-docs/inception/measurements/measurement-plan.md`.

Include tables for:

- Known values
- Measured values
- Assumed values
- Missing values
- Measurement method
- Impact if wrong

## Rule

No dimension should be treated as final unless it is either measured, user-confirmed, or intentionally parameterized with a validation plan.
