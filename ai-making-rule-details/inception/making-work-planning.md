# Making Work Planning

Purpose: define the sequence for turning the design into a buildable package.

## Plan

Define:

- Deliverables
- File outputs
- Work sequence
- Measurements needed
- Test coupon scope
- Prototype sequence
- Review gates
- Owner/role assumptions
- Risks and mitigations
- Definition of done

Use the quality gate model when defining review gates:

- Gate 0: problem definition
- Gate 1: material and fabrication decision
- Gate 2: measurement readiness
- Gate 3: fabrication structure review
- Gate 4: CAD/CAM export validation
- Gate 5: test coupon pass
- Gate 6: prototype pass
- Gate 7: beginner buildability pass
- Gate 8: release package pass

## Output

Create `making-docs/inception/plans/making-execution-plan.md`.

Include:

- Recommended workflow
- Stage execute/skip decisions
- Work breakdown
- Gate criteria
- Gate owner or evidence source
- Blocked-gate handling
- Validation commands or methods
- Next action
