# User And Scenario Analysis

Execute when the object will be used repeatedly, assembled by a novice, shared, sold, or operated in a real environment.

## Analyze

- Builder skill level
- User skill level
- Assembly environment
- Use environment
- Normal use sequence
- Misuse cases
- Maintenance or repair
- Storage and transport
- Ergonomic constraints
- Accessibility or one-handed operation

## Output

Create `making-docs/inception/scenarios/user-scenarios.md`.

Include:

- Primary builder scenario
- Primary user scenario
- Likely mistakes
- Acceptance expectations
- Design implications
