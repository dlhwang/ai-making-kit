# CAD/CAM Deliverable Design

Purpose: decide what design files and export modes are required.

## Define

- Source CAD format
- Output formats: DXF, SVG, STL, STEP, PDF, PNG, G-code, etc.
- View/mode naming
- Units
- Coordinate orientation
- Individual part views
- Assembly preview
- Fit check view
- Test coupon view
- Beginner/novice layout, if needed
- Export commands
- Validation commands

## Output

Create `making-docs/execution/design/cad-cam-deliverable-design.md`.

Include:

- Required files
- Required modes/views
- Export matrix
- Validation matrix
