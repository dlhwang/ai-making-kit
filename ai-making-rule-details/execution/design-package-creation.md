# Design Package Creation

Purpose: create or revise the physical design package.

## Rules

- Preserve original source files unless the user explicitly asks for replacement.
- Prefer parameterized dimensions.
- Separate preview geometry from fabrication geometry.
- Separate 2D manufacturing layouts from 3D assembly previews.
- Include fit-check geometry when real objects interface.
- Include beginner-friendly modes when novice building matters.
- Avoid hiding critical dimensions in comments only.

## Output

Create deliverables in project-appropriate paths and update:

- `making-docs/execution/deliverables/deliverable-index.md`

Include:

- File name
- Purpose
- Audience
- Status
- Validation needs
