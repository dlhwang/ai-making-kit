# Review And Validation

Purpose: verify that the design package is buildable and aligned with requirements.

## Validate Against

- Making requirements
- Material and fabrication analysis
- Measurement plan
- Measured specs, when available
- CAD/CAM deliverable design
- Quality criteria
- Quality gate review, when readiness is claimed
- Test coupon plan
- Test coupon report, when a coupon was physically made
- Assembly guide
- Beginner buildability, if relevant

## Output

Create `making-docs/execution/review/review-summary.md`.

Include:

- Overall status: pass, pass with issues, or fail
- Requirement coverage
- Export validation
- Measurement coverage
- Test coupon coverage
- Quality gate status
- Buildability risks
- Usability risks
- Recommended fixes
- Release recommendation
