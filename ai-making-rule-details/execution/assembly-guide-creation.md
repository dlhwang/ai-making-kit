# Assembly Guide Creation

Purpose: make the design buildable by another person.

## Include

- Required tools
- Required materials
- Part ID table
- Part quantities
- Layer/stacking counts
- Orientation: front/back/top/bottom/left/right
- Adhesive or fastener locations
- Drying/wait times
- Fit check steps
- Warnings
- Failure adjustment table

## Output

Create an assembly guide, usually:

- `making-docs/execution/guides/assembly-guide.md`

or a project-specific guide path requested by the user.
